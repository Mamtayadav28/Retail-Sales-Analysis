# 📊 Retail Sales Data Analysis using Python, NumPy, Pandas & SQL

## 📌 Project Overview

This project analyzes retail sales data using Python and SQL to generate meaningful business insights. It covers the complete data analysis workflow including data cleaning, feature engineering, exploratory data analysis (EDA), statistical analysis with NumPy, SQL queries using SQLite, and data visualization with Matplotlib.

The objective of this project is to demonstrate practical data analytics skills on a real-world retail dataset.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* SQLite
* Jupyter Notebook

---

## 📂 Dataset

**Dataset:** Online Retail Dataset (Sample)

Online Retail dataset (around 8,000 rows before cleaning) 

The dataset contains customer transactions from an online retail store, including invoice details, products, quantities, prices, customers, and countries.

## 📋 Project Workflow

### 1. Data Loading

* Imported CSV using Pandas
* Explored dataset structure
* Checked data types and dimensions

### 2. Data Cleaning

* Removed duplicate records
* Handled missing values
* Removed cancelled orders
* Removed invalid quantities
* Verified data quality

### 3. Feature Engineering

Created new columns including:

* Revenue
* Profit Percentage (if applicable)
* Discount Category (if applicable)
* Order Month
* Month Number
* Order Day
* Order Hour
* Weekend Order

### 4. Exploratory Data Analysis

Business questions answered:

* Total Revenue
* Total Orders
* Total Customers
* Top Selling Products
* Top Customers
* Revenue by Country
* Monthly Revenue Trend
* Revenue by Weekday
* Revenue by Hour
* Average Order Value (AOV)
* Repeat Customers
* Top Revenue Invoices

### 5. NumPy Analysis

Calculated:

* Mean
* Median
* Maximum
* Minimum
* Standard Deviation
* Variance
* Quartiles
* Outlier Detection using IQR

### 6. SQL Analysis

Performed 20 SQL queries including:

* Revenue Analysis
* Customer Analysis
* Product Analysis
* Country-wise Analysis
* Monthly Analysis
* Order Analysis
* Aggregate Functions
* GROUP BY
* HAVING
* Subqueries

### 7. Data Visualization

Created visualizations using Matplotlib:

* Top Products by Revenue
* Revenue by Country
* Top Customers
* Monthly Revenue Trend
* Revenue by Weekday
* Revenue by Hour
* Revenue Distribution Histogram
